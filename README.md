# Relational Database Management System

This repository contains small projects developed during the study of **Relational Database Management System**.

## 📚 Repository Structure

Projects are organized into `Part` folders, each corresponding to a specific topic.

```text
relational-database-management-system/
│
├── Part01-SQL/
├── Part02-DB Design/
├── Part03-Programming/
```

## 🎯 Objectives

This repository was created to:

* Learn and practice Relational Database Management System (RDBMS).
* Store projects organized by learning topics.
* Track learning progress and improve database skills.
* Build a solid foundation for larger databases in the future.

## 🛠️ Technologies Used

* SQL Server
* Visual Paradigm
* Git & GitHub

## 📖 Learning Topics

The following topics are covered in this repository:

* SQL (Structured Query Language)

  * Basic Select

    * Select only
    * Select one table
    * Select Distinct
    * Select Order By
    * Select Where
    * Select Range (Between & In)
    * Select Null
    * Select Like
  * Sub query
    
    * Single value
    * Multiple values
    * All & Any
  * Aggregate Funtions

    * Count
    * Min, Max, Sum, Avg
  * Join

    * Cartesian
    * Inner
    * Outer (Left Join, Right Join, Full Join)
* Database Design

  * Design Principles

    * Overview
    * Anomalies
    * Mantras of Design
    * Structure view
  * Relationship

    * One table
    * Two tables
* Programming

  * Variables
  * View
  * Function & Procedure
  * Trigger
